# Support

Thanks for using **Cypherock COT**.

## Getting help

- For usage questions and troubleshooting, open a GitHub issue.
- For feature proposals, open a GitHub issue with the `enhancement` label.
- For security concerns, please follow `SECURITY.md` and report privately.

## Before opening an issue

Please include:

- your environment (OS, compiler, Node.js version),
- exact steps to reproduce,
- expected vs actual behavior,
- and relevant logs from both server and client.

This helps maintainers debug quickly and saves everyone a few gray hairs.
